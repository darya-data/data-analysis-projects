# Прогнозирование процентной ставки по депозитам овернайт

## Задача

Возьмите реальный временной ряд из открытого источника (например, данные о продажах товара или курсе валюты). Проведите разведочный анализ: постройте график ряда и его автокорреляционную функцию (ACF/PACF), проверьте ряд на стационарность (тест Дики-Фуллера). При необходимости примените дифференцирование для стабилизации дисперсии и/или логарифмирование. На основе ACF/PACF подберите порядки (p,d,q) для модели ARIMA. Обучите модель и постройте график с фактическими и предсказанными значениями. По желанию для сравнения можете использовать нейросетевую архитектуру.

## Результаты

В ходе исследования был проведен анализ временного ряда процентных ставок по депозитам овернайт, содержащий ежедневные значения процентной ставки с 3.02.2014 по 28.03.2025. 

Этапы анализа:
- Сбор данных с сайта ЦБ РФ посредством Selenium и формирование датафрейма.
- Первичный анализ данных и выделение компонентов временного ряда.
- Проверка гипотез на стационарность и наличие тренда. Проверка ряда на стационарность проводилась при помощи теста Дики-Фуллера: исследуемый ряд является нестационарным. Проверка ряда на наличие тренда проводилась при помощи теста Квятковского-Филлипса-Шмидта-Шина: ряд не подразумевает тренд.
- Для прогноза значений использованы:
  - ARIMA - были построены несколько вариантов моделей, но адекватных предсказаний с их помощью получить не удалось.
  - LSTM – рекомендована для дальнейшего использования, так как при помощи модели удалось получить адекватные прогнозы процентной ставки.
