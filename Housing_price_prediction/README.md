# Прогноз цен на недвижимость

## Задача

Задача заключается в том, чтобы спрогнозировать стоимость квартир. Показателем оценки является R2 коэффициент детерминации (sklearn.metrics.r2_score), необходимо получить $R^2 > 0.6$.

## Результаты

В процессе исследования было выполнено следующее:
- Проведен разведочный анализ данных, который выявил проблемы пропусков, выбросов и несогласованности данных.
- Были реализованы и оценены следующие базовые алгоритмы:
  - LinearRegression.
  - GammaRegressor.
  - RandomForestRegressor.
  - LGBMRegressor.
  - CatBoostRegressor.
  
  Линейная и гамма-регрессия показали самые плохие результаты. Наилучший результат показала модель CatBoostRegressor.
- Осуществлена генерация новых признаков, благодаря чему улучшено качество CatBoostRegressor.
- Итоговая модель имеет значение $r^2=0.757$ на тестовых данных.
