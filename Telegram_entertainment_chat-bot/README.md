# Разработка многофункционального развлекательного чат-бота

## Задача

Создать чат-бот для Telegram с применением машинного обучения. 

## Результаты

Entertainment_chat-bot в Telegram - это многофункциональный развлекательный чат-бот, разработанный с нуля. Проект сочетает обработку естественного языка, машинное обучение и механики взаимодействия с пользователем для создания увлекательного пользовательского опыта. Бот реализует несколько полезных функций, предоставляя возможность для развлечений и поиска информации.

Основные возможности чат-бота: 
- Общение: бот поддерживает диалог с учетом смысла предыдущих сообщений. Разговоры сохраняются с целью анализа и улучшения. Для генерации ответов использована легкая русскоязычная модель 'sberbank-ai/rugpt3small_based_on_gpt2', так как она хорошо работает на CPU, дает естественные диалоги и учитывает контекст введенного текста.
- Поиск данных: пользователь может осуществить быстрый поиск информации благодаря интеграции с Википедией через Wikipedia-API. Введенный пользователем текст очищается и форматируется. Также существует возможность перейти по ссылке в интернет-поисковик. 
- Подбор фильмов: реализован поиск фильмов по запросу (можно указать краткое описание, жанр, актеров). Пользователь описывает, какой фильм хочет, а бот находит самые похожие. Для семантического поиска использованы Word2Vec и Annoy. Для рекомендаций загружен и предобработан датасет «Топ-250» Кинопоиска — проверенная база популярных фильмов. 
- Генератор случайных анекдотов и мемов: их парсинг осуществляется с популярных сайтов, находящихся в открытом доступе. 
- Игра "Камень-ножницы-бумага": реализован интуитивно понятный  интерфейс популярной игры с кнопками выбора. 
- Шуточные прогнозы на день: предсказания хранятся в виде 4 текстовых файлов. Система предсказаний комбинирует случайные фразы из них, создавая уникальные, но согласованные прогнозы. 
- Чат-бот автоматически сохраняет диалоги с каждым пользователем, а также логирует запросы по поиску в Википедии и по рекомендации фильмов. 

Как работает бот (пользовательский сценарий):
- Пользователь запускает бота командой /start.
- Видит главное меню со следующими опциями: 
    - Найти информацию (перейти в Яндекс или найти информацию в Википедии). 
    - Посмеяться (получить анекдот или получить мем).
    - Порекомендовать фильм.
    - Поиграть с игру (играть в игру «Камень, ножницы, бумага» или получить предсказание на день). 
- Пользователь выбирает нужную функцию через инлайн-кнопки. 
- Пользователь получает интерактивный ответ с возможностью продолжения или выхода в главное меню. 
- Если конечный выбор действия в меню еще не осуществлен, пользователь может вступить в диалог с ботом, написав любое сообщение. 

Проект разделен на 8 модулей, каждый из которых отвечает за конкретный функционал:
- main.py - основной запуск бота и инициализация.
- anecdotes.py - парсинг анекдотов.
- films.py - семантический поиск фильмов по запросу пользователя с использованием Word2Vec и Annoy для быстрого поиска ближайших соседей с логированием.
- game.py - реализация игры "Камень-ножницы-бумага". 
- horoscopes.py - генерация предсказания на день. 
- memes.py - парсинг мемов.
- talking.py - генерация контекстно-зависимых ответов на сообщения пользователя с использованием языковой модели и сохранением истории диалога.
- wiki.py – поиск и обработка статей из Wikipedia с очисткой текста и логированием.
 
Этот проект - практический пример интеграции различных технологических стеков (машинное обучение, парсинг данных и внешние API) в единое функциональное приложение. Проект является полностью рабочим, может быть масштабирован, и может служить основой для более сложных развлекательных или образовательных платформ.
