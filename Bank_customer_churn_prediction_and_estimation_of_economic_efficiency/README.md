# Прогнозирование оттока клиентов банка и расчет экономической эффективности

## Задача

Предоставлены данные с оттоком клиентов банка. Необходимо спрогнозировать, уйдет клиент или нет.

Провести эксперименты с моделями:
- логистическая регрессия;
- бустинг.

Обучить 2-3 модели, подобрать гиперпараметры.

Рассчитать экономическую эффективность, исходя из трат бизнеса:
- 1 доллар - на удержание клиента;
- 2 доллара - с каждого правильно классифицированного (True Positive).

Сравнить модели по метрикам и показателю экономической эффективности.

## Результаты

В ходе работы было выполнено следующее:
- Проведен разведочный анализ данных.
- Обучены базовые модели логистической регрессии и градиентного бустинга. Осуществлен подбор гиперпараметров при помощи GridSearchCV, и подобран порог наилучшего разделения классов. Модель градиентного бустинга получила наилучшую метрику roc_auc=0.857.
- Выполнен расчет экономической эффективности для каждой модели: базовая модель логистической регрессии показала отрицательную экономическую эффективность (-101), логистическая регрессия с подбором параметров сработала чуть лучше, но все равно отрицательно(-48). Градиентный бустинг с подбором параметров наиболее финансово обоснован: получили прибыль 120. Данная модель рекуомендуется для использования.
