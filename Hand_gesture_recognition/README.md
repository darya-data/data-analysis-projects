# Распознавание жестов рук

## Задача

Нужно написать приложение, которое будет считывать кадры с
веб-камеры. В процессе считывания человек показывает жесты руками, а алгоритм должен считать их.

Для распознавания жестов скачать датасет https://www.kaggle.com/gti-upm/leapgestrecog, разработать модель для обучения и  обучить эту модель.

## Результаты

- В работе был использован датасет для распознавания жестов рук, состоящий из набора изображений в ближнем инфракрасном диапазоне, полученных с помощью датчика Leap Motion.  
- Была выполнения аугментация данных для увеличения их разнообразия.
- Для обучения использовалась модель resnet50 с предобученными весами.  Модель была проверена на новых фотографиях, сделанных при помощи веб-камеры. Модель правильно распознала 3 из 4 фотографий. Это хороший результат. Возможно, для его улучшения необходимо более длительное обучение.
