# Репозиторий проектов по анализу данных

Данные проекты охватывают практику анализа данных с использованием статистического анализа и машинного обучения.  Репозиторий включает коллекцию практических проектов по анализу табличных данных, прогнозированию временных рядов, обработке естественного языка, компьютерному зрению. Список проектов с описанием, используемым стеком и ссылками на код представлен в таблице ниже.

| № | Название проекта | Описание | Используемые библиотеки | 
|---|---------|--------------|-------------|
| 1 | [Прогнозирование факта невыполнения обязательств по кредиту](./Prediction_a_credit_default) | Предоставлены данные о финансовом положении и кредитных обязательствах заемщиков банка. Необходимо построить модель для прогнозирования невыполнения долговых обязательств по текущему кредиту.| catboost, pandas, sklearn, imblearn| 
| 2 | [Прогнозирование оттока клиентов банка и расчет экономической эффективности](./Bank_customer_churn_prediction_and_estimation_of_economic_efficiency) | Предоставлены данные с оттоком клиентов банка. Необходимо спрогнозировать, уйдет клиент или нет, и рассчитать экономическую эффективность моделей, исходя из трат бизнеса на удержание клиента.  | pandas, sklearn |
| 3 | [Прогнозирование процентной ставки по депозитам овернайт](./Overnight_deposit_rate_forecasting) | Задача состоит в прогнозировании значений реального временного ряда. Для исследования использован временной ряд процентных ставок по депозитам овернайт, содержащий ежедневные значения процентной ставки с 3.02.2014 по 28.03.2025. | selenium, pandas, sklearn, torch, statsmodels | 
| 4 | [Сравнительный анализ методов uplift-моделирования для оптимизации маркетинговых коммуникаций](./Comparative_analysis_of_uplift_modeling_methods_for_optimizing_marketing_communications) | Дан набор данных маркетинговой кампании. Необходимо провести uplift-моделирование тремя способами: одна модель с признаком коммуникации, модель с трансформацией таргета,  две независимые модели.  | catboost, pandas, sklearn, sklift |
| 5 | [Прогнозирование оттока клиентов новостного портала](./News_portal_clients_churn_prediction) | Доступны данные о пользователях новостного портала и прочитанных ими новостями. Необходимо построить модель прогнозирования оттока. Оттоком считается, когда клиент перестал посещать сайт.  | pandas, nltk, gensim, sklearn | 
| 6 | [Разработка многофункционального развлекательного чат-бота](./Telegram_entertainment_chat-bot) | Создание чат-бот для Telegram с применением машинного обучения. | telebot, nltk, gensim, annoy, beautifulsoup | 
| 7 | [Прогноз цен на недвижимость ](./Housing_price_prediction) | Задача заключается в том, чтобы спрогнозировать стоимость квартир по их характеристикам. Показателем оценки является коэффициент детерминации. | catboost, pandas, sklearn, shap | 
| 8 | [Предсказание наличия сердечно-сосудистых заболеваний ](./Prediction_of_cardiovascular_diseases) | Необходимо предсказать наличие сердечно-сосудистых заболеваний по результатам классического врачебного осмотра. Нужно сравнить полученные модели по метрикам классификации и по метрикам бизнеса. | catboost, xgboost, sklearn | 
| 9 | [Многоклассовая классификация заболеваний сои с применением методов снижения размерности и подбора гиперпараметров](./Multiclass_classification_of_soybean_diseases_using_dimensionality_reduction_and_hyperparameter_tuning) | Необходимо использовать датасет заболеваний сои для решения задачи многоклассовой классификации. | 10 | [Сравнительный анализ  нейросетевых архитектур при решении задачи многоклассовой классификации](./Comparative_analysis_of_neural_networks_for_multiclass_classification) | Необходимо взять датасет текстовых отзывов за лето и сравнить нейросетевые архитектуры при решении задачи многоклассовой классификации. | pandas, keras, nltk, sklearn | 
| 11 | [Распознавание жестов рук](./Hand_gesture_recognition) | Представлена база данных распознавания жестов рук. Нужно написать приложение, которое будет считывать кадры с веб-камеры и выводить класс, который распознает модель. | torch | 
| 12 | [Исследование симптомов ПТСР у пациентов с раком яичников ](./PTSD_symptoms_in_ovarian_cancer_patients) | Необходимо проанализировать датасет, в котором собрана информация о пациентах с раком яичников и их психологическом состоянии. Сформулировать гипотезы, построить линейную модель зависимости симптомов ПТСР от количественных предикторов,  провести дисперсионный анализ зависимости PTSR symptomes от стадии рака. | R: tidyverse, car, multcomp |
